Integrated Calendars

Seamless scheduling for teams and individuals.

Stalwart brings powerful calendar and scheduling capabilities directly into your collaboration stack—no add-ons or third-party tools needed. Plan events, manage recurring meetings, set alerts, and coordinate with teammates using open protocols designed for real-time performance and seamless integration. With support for both JMAP and CalDAV, your calendars stay perfectly in sync across all clients and devices. Whether you're working solo or across a distributed team, Stalwart ensures you stay on schedule, informed, and in control.

  • Full-featured calendaring.
  • Scheduling tools for meetings, tasks, and recurring events.
  • Real-time alerts and reminders.
  • Integrated free/busy availability lookup.
  • Seamless access via CalDAV and JMAP protocols.
  • Support for group and shared calendars.

Unified contacts

Manage, share, and sync your contacts.

Stalwart offers seamless contact management as part of its integrated collaboration features. Designed for compatibility and performance, it supports both JMAP and CardDAV, allowing users to access, sync, and manage contacts across any compliant client or device. Whether you're handling personal contacts, shared address books, or organizing groups, Stalwart ensures your data is consistent, reliable, and ready for real-time collaboration—without the complexity of external services.

  • Full contact management.
  • Support for group and shared address books.
  • Real-time sync via JMAP and CardDAV protocols.

Integrated File Access

Secure, centralized access to your documents.

Stalwart includes built-in file storage capabilities, making it easy to manage and share documents within the same secure platform that powers your mail and collaboration tools. With support for WebDAV and the JMAP for File Storage extension, users can store, retrieve, and sync files across devices and clients effortlessly. Whether you're managing attachments, documents, or shared team resources, Stalwart provides a unified, protocol-driven solution that keeps your data accessible and protected.

  • Built-in file storage.
  • Access via WebDAV and JMAP for File Storage.
  • Support for user, group and shared file spaces.
  • Secure file sharing with fine-grained permissions.

Flexible sharing

Granular control over calendars, contacts, and files.

Stalwart makes collaboration seamless with built-in sharing capabilities across calendars, address books, and file storage. Whether you're sharing with a single user, a group, or an entire organization, Stalwart gives you precise control over what others can see or do. With support for WebDAV ACLs and the JMAP for Sharing extension, users can assign granular permissions to manage access with confidence—no manual configuration or external tools required.

  • Support for user, group, and shared resources.
  • Share calendars, address books, and file spaces.
  • Fine-grained permission control for read, write, and manage access.
  • Based on WebDAV ACL and JMAP for Sharing standards.
